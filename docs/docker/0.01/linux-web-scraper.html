1. How to make a webscraper with bash on ubuntu

Web Scraper
URL=""http://www.bbc.com""
SELECTOR="div.media"
OUTPUT="allMedia"

echo ${URL} \
  | wget -O- -i- \
  | hxnormalize -x \ (1)
  | hxselect -i ${SELECTOR} \ (2)
  | lynx -stdin -dump > ${OUTPUT} (3)

bat --language man ${OUTPUT}
1 hxnormalize → pretty-print an HTML file
2 hxselect → extract elements or attributes that match a (CSS) selector -i: case insensitive
3 lynx → a general purpose distributed information browser for the World Wide Web
